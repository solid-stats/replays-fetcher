<objective>
Move the three cross-band data-contract DTOs into the leaf `src/types/` module and make `src/types/run-summary.ts` truly leaf — a pure, behavior-preserving type-move (ARCH-01).

The four cross-band contracts named by the phase are `ReplayCandidate`, `RawReplayStorageEvidence`, `RunSummary`/`CompactRunSummary`, and `IngestStagingPayload`. `RunSummary`/`CompactRunSummary` already live in `src/types/run-summary.ts` [src: src/types/run-summary.ts#L51,L83] — but that file imports UPWARD from `discovery/`, `staging/`, and `storage/` [src: src/types/run-summary.ts#L1-8], so the contracts module is not actually leaf. Moving the three remaining DTOs down (plus the two result wrappers `run-summary.ts` consumes) fixes ARCH-01 *and* makes `run-summary.ts` leaf in one move.

Purpose: Cross-band contracts at the bottom of the dependency graph is the prerequisite for every later v3.1 phase — it removes the latent upward import so Phase 21's bulk `interface→type` conversion creates no new upward import.
Output: Three new `src/types/*.ts` contract files, an updated leaf `run-summary.ts`, and downward re-export shims in the three band `types.ts` files so call sites don't churn.
</objective>
