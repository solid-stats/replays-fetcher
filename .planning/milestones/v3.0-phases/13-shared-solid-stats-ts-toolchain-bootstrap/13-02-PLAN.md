  1. Создай аннотированный тег: `git -C "$WORKDIR" tag -a v0.1.0 -m "Initial toolchain presets: tsconfig/oxlint/oxfmt/vitest + lefthook"`. (Тег по умолчанию указывает на текущий HEAD = зелёный SHA.)

  3. Подтверди, что тег виден в origin и указывает на зелёный SHA:

     - peeled SHA тега (`git rev-list -n1 v0.1.0`) равен `HEAD_SHA`.
  4. (Опционально) подтверди через `gh`: `gh api repos/solid-stats/ts-toolchain/git/refs/tags/v0.1.0` возвращает объект тега.

  </action>

  </verify>
  <done>Аннотированный тег `v0.1.0` существует в `origin`, peelится в зелёный master HEAD SHA, и виден через `git ls-remote --tags`.</done>
